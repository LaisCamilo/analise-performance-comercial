# Análise de Dados de Vendas

# Objetivo
Analisar dados de vendas para gerar insights de negócio e apoiar a tomada de decisão orientada por dados.

# Principais Análises
Produtos: Copiadoras e impressoras são mais lucrativas; mesas e impressoras 3D geram prejuízo.
Categorias: Copiers, Phones e Accessories se destacam; Tables e Bookcases têm baixo desempenho.
Sazonalidade: Picos no fim do ano e períodos de baixa ao longo do ano.
Clientes: Destaques variam por ano → foco em retenção.
Segmentos: Consumer lidera em volume, mas deve ser analisado com lucro.

# Insights
Priorizar produtos mais lucrativos.
Revisar ou remover itens com prejuízo.
Usar sazonalidade para planejar campanhas.
Investir em retenção de clientes.
Focar nos segmentos mais relevantes.
